# Hangman Game 🎮

A simple **Hangman Game** implemented in C++ for beginners.  
This project demonstrates the use of strings, loops, conditionals, and vectors in C++.

---

## 📌 Features
- Word guessing game with underscores for hidden letters.
- Tracks wrong guesses and limits attempts.
- Prevents repeated guesses.
- Displays win/lose messages at the end.

---

## 🛠️ Technologies Used
- **C++** (Standard Library)
- Basic concepts: loops, conditionals, strings, vectors
